# Contributing to DevFlow

Thank you for your interest in contributing to DevFlow! We appreciate your help in making this project even better.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Clone Locally**: `git clone https://github.com/YOUR_USERNAME/DevFlow.git`
3.  **Create a Branch**: `git checkout -b feature/amazing-feature`
4.  **Make Changes**: Implement your feature or fix your bug.
5.  **Run Tests**: Ensure all tests pass. `cd backend && pytest`
6.  **Commit Changes**: `git commit -m 'Add some amazing feature'`
7.  **Push to Branch**: `git push origin feature/amazing-feature`
8.  **Open a Pull Request**: Submit your changes for review.

## Code of Conduct

Please be respectful and helpful to all contributors and users.

## Reporting Issues

Use the GitHub Issues tracker to report bugs or request features. Please provide as much context as possible.
